# Releases

Pre-built installers for WishperType will be published here for tagged releases.

| Platform | File |
|----------|------|
| macOS (Apple Silicon) | `WishperType_*_aarch64.dmg` |
| macOS (Intel) | `WishperType_*_x64.dmg` |
| Windows | `WishperType_*_x64-setup.exe` |
| Linux | `WishperType_*_amd64.AppImage` |

> This folder is intentionally empty in the source repo. Release artifacts are attached to [GitHub Releases](https://github.com/utpalsinghdev/wishper-type/releases).
