# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a vulnerability

**Please do not open a public issue for security vulnerabilities.**

Instead, report them privately:

1. Open a [GitHub Security Advisory](https://github.com/utpalsinghdev/wishper-type/security/advisories/new) (recommended), or
2. Email the maintainer via their GitHub profile: [@utpalsinghdev](https://github.com/utpalsinghdev)

Include:

- Description of the vulnerability
- Steps to reproduce
- Impact (what an attacker could do)
- Your OS and WishperType version if relevant

You should receive a response within 7 days. We will work with you on a fix before any public disclosure.

## Scope

This policy covers the WishperType desktop application and its bundled transcription server.
